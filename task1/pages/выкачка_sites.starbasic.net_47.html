Форум інформатиків України

Оперативна допомога по будь-яких питаннях, пов’язаних з інформатикою, комп’ютерами, роботою в школі, придбанням якогось пристрою. Цікаві опитування. Мова програмування для школи. Інформаційні технології у розвитку освіти та суспільства. Проблеми викладання інформатики. Термінологія в шкільній інформатиці. Юридично-правові аспекти. Кабінет інформатики: документація, стенди, зовнішній вигляд, потужність комп’ютерів та інше. Тестуючі системи. Особливості викладання інформатики у ВНЗ, обговорення фахових питань. Інноваційні проекти МОН. Вплив комп’ютера на здоров’я людини. Безпека роботи в Інтернеті. Змагання, конкурси, олімпіади. Атестація вчителя інформатики. Новини освіти. Інформатика за кордоном.

Авторами сайту є команда вчителів інформатики України, які об’єдналися заради спільної мети: підняти престиж предмету "Інформатика" на державному рівні, виробити стратегію його подальшого розвитку.

Пріорітетними завданнями Форуму інформатиків України є:
Обмін досвідом між форумчанами.
Вирішення різних проблем, що виникають в роботі.
Обговорення підручників та посібників, різноманітної методичної літератури з інформатики.
Обмін творчими надбаннями.
Обговорення пcихолого-педагогічних аспектів викладання інформатики.
Обговорення програм з інформатики для інваріантної складової навчальних планів та для курсів за вибором / факультативів.
Висвітлення новинок програмного та апаратного забезпечення та багато іншого корисного і цікавого.

Сервис онлайн тестирования Quizful

Проект Quizful предназначен для онлайн-тестирования специалистов в сфере информационных технологий. Quizful является community-driven сервисом и развивается во многом благодаря усилиям сообщества ИТ-специалистов.

На Quizful вы найдете более 140 тестов:

  • тесты по администрированию (Unix, Linux, Windows, MacOS)
  • тесты по программированию (C++, Java, C#, 1C, PHP, Python, Ruby)
  • тесты по базам данных (SQL, Oracle, MS SQL, MySql)
  • тесты по веб-технологиям (HTML, CSS, JavaScript, HTTP, jQuery)
  • тесты по управлению проектами (экстремальное программирование, Scrum)

Сервис Quizful предназначен для ИТ-специалистов и позволяет с помощью онлайн тестов проверить себя в различных областях информационных технологий, подготовиться к аттестации или собеседованию, систематизировать свои знания или получить новые знания по интересующей теме.

Сервис является обучающим и направлен на помощь ИТ сообществу, после прохождения тестов можно просмотреть правильные ответы с объяснениями. Сервис постоянно наполняется новыми тестами и расширяется за счет добавления новой функциональности.

Quizful предлагает пользователям возможность активно участвовать в развитии сервиса: составлять вопросы, оставлять отзывы и комментарии, голосовать за качество тестов, участвовать в конкурсах и опросах.

Просмотрщик электронных документов

Программа поможет вам просмотреть электронные документы в самых разных форматах. Мы попытались сделать так, чтобы вам больше не надо было, получив документ на компьютер, искать, чем же его открыть.

Наш просмотрщик поддерживает большинство форматов, используемых в современном компьютерном мире. В первую очередь это, конечно, PDF и многостраничный TIFF. А также DjVu, JBIG2, WWF и XPS. Также вы можете использовать нашу программу, чтобы просматривать графические файлы BMP, PCX, JPEG, GIF, PNG, DCX, WMF, EMF, PSD. Любителям электронных книг, программа поможет прочитать книги в форматах FB2 (в том числе FB2.ZIP), TXT, TCR, PalmDoc(PDB), MOBI, AZW, EPub. Для тех, кому нравятся комиксы, мы добавили поддержку форматов CBR и CBZ.

Просмотрщик имеет широкий набор функций. Масштабирование, поворот страниц настройка яркости и контрастности для документа, выбор удобного для Вас способа расположения страниц (по одной или разворотом).

В коротком описании, сложно рассказать обо всех возможностях, которые предоставляет наш просмотрщик. Мы не упомянули о возможности сформировать набор цитат для документа, просмотре обложек книг в папке на компьютере и т.п. Просто скачайте и попробуйте. Мы предоставляем полностью функциональную версию совершенно бесплатно. Для домашнего, образовательного и другого не коммерческого использования регистрация не требуется. вы можете даже не устанавливать ничего на свой компьютер: скачайте портативную версию нашей программы, разархивируйте и испытайте.

PHP.SU

Все о PHP. Уроки, статьи, скрипты, дистрибутивы, форум и многое другое. Целью создания проекта PHP.SU было желание удовлетворить потребность начинающих веб-программистов в интернет-ресурсе с богатым тематическим содержанием. Сейчас портал PHP.SU способен практически полностью удовлетворить потребности начинающих программистов языка PHP в материалах по PHP, MySQL, в необходимых вспомогательных инструментах, таких как редакторы PHP кода и сопутствующие утилиты для PHP, справочнике по функциям PHP на русском языке, системы поиска необходимой информации, форуме разработчиков и.т.п. Продвинутые и профессиональные веб-программисты, несомненно, также найдут для себя полезную информацию на портале PHP.SU.

Вы начинаете изучать PHP? Ознакомьтесь с вводным курсом обучения PHP. Так же для новичков наши профессионалы собрали материалы по форуму PHP.SU и разместили это в виде Уроков PHP.

Знаете основы PHP и хотите укрепить свои познания? Cправочник по всем функциям PHP, полностью на русском. Справочник по функциям PHP оснащен системой поиска функций по ключевым словам.

В PHP есть обширные средства для работы с СУБД, такими, как MySQL. Вы можете использовать PHP для работы с большим количеством различных типов СУБД. Наш портал содержит множество статей по PHP, MySQL, Apache, PEAR, регулярным выражениям, XML, другим веб-технологиям.

У нас Вы найдете большое количество учебников и справочников по PHP, MySQL, HTML, XML, JavaScript и.т.д.

Раздел "PHP скрипты" поможет Вам скачать большое количество различных скриптов PHP. Раздел "Download" поможет Вам скачать необходимые компоненты: PHP, Apache, MySQL, PECL, PEAR, редакторы PHP кода, полезные утилиты для PHP и документацию по PHP, MySQL, PEAR и Apache.
